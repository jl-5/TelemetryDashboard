# TelemetryDashboard

A real-time metrics dashboard built with React, TypeScript, and Apache ECharts. This project simulates production telemetry systems and demonstrates realistic frontend architecture, time-series visualization, and production-style deployment using Docker and Nginx.

---

## Live Demo

https://jl-5.github.io/TelemetryDashboard/

---

## Project Overview

This dashboard simulates a monitoring interface similar to tools like Grafana or Datadog. It polls a mock API, converts metric snapshots into time-series data, and visualizes them in real time.
